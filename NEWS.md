# vbracket 1.3.0

* Added `bracket_layer_spacing` parameter to `legend_bracket()`, `draw_legend_with_brackets()`, and `create_annotation_legend_grob()` for manual control of horizontal spacing between bracket layers.

# vbracket 1.2.0

* Added `line_length`, `line_width`, and `item_spacing` parameters to `legend_bracket()` and `draw_legend_with_brackets()` for manual override of auto-scaled legend symbol dimensions.

# vbracket 1.1.0

* Added proportional text scaling: legend symbol size and item spacing now scale automatically with `text_size`.

# vbracket 1.0.2

* Fixed Rd examples and resolved CRAN policy issues for initial CRAN release.

# vbracket 1.0.0

* Initial release.
